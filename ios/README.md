# beneon (iOS)

SwiftUI iOS application for beneon.

## Requirements

- Xcode 16+
- iOS 17.0+
- [XcodeGen](https://github.com/yonaskolb/XcodeGen)

## Getting Started

```bash
cd ios
xcodegen generate
open beneon.xcodeproj
```

Select your development team in the project settings, then build and run.
